## ⚡ 快速开始

### 基本使用（无需任何改动）
```tsx
import { MarkdownRenderer } from '@v2/markdown-parser';

<MarkdownRenderer content={content} />
```

### 使用新密度模式（推荐）
```tsx
// 主要内容 - 舒适阅读
<MarkdownRenderer content={content} density="comfortable" />

// 侧边栏/卡片 - 紧凑布局
<MarkdownRenderer content={content} density="compact" />
```

## ✨ 核心改进

1. **⬇️ 降低 CSS 特异性** - 使用 `:where()`，零 `!important`
2. **🎨 修复段落透明度** - 不再影响子元素对比度
3. **📏 密度模式** - `comfortable` (1.75) / `compact` (1.45)
4. **📱 移动端表格** - 自动横向滚动，不破坏布局
5. **⌨️ 键盘可访问性** - 清晰的焦点指示
6. **🎭 渐进增强** - oklch 颜色 + rgba 回退
7. **🔗 锚点导航** - 不被固定头部遮挡
8. **♿ 减少动效** - 尊重用户偏好
