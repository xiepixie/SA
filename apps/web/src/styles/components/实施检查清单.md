## 💡 快速问题排查

### 问题：密度模式不生效
**检查：**
1. 是否使用了最新的 `MarkdownRenderer` 组件？
2. `data-density` 属性在 DOM 中是否正确？
3. CSS 文件是否正确导入？

### 问题：表格在移动端仍然破坏布局
**检查：**
1. `.md-table-wrap` 类是否自动添加？
2. 是否使用了最新版本的解析器？
3. CSS 中 `.md-table-wrap` 样式是否存在？

### 问题：锚点仍被头部遮挡
**检查：**
1. `--md-header-offset` 值是否与实际头部高度匹配？
2. `scroll-margin-top` 是否应用到标题元素？
3. 浏览器是否支持 `scroll-margin-top`？（98%+ 支持）

### 问题：样式难以覆盖
**检查：**
1. 是否使用了 `:where()` 版本的 CSS？
2. 是否仍在某处使用 `!important`？
3. 特异性计算是否正确？
